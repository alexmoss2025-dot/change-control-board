About This Documentation Site

This site serves as the central hub for Moss Energy's Change Control Board (CCB) documentation suite. All documents are interactive HTML mockups designed to demonstrate proposed system interfaces, workflows, and processes.

Each document has been crafted to provide a realistic preview of the final system, enabling stakeholders to review and provide feedback before development begins.

Use the navigation cards above or the top menu bar to access any document. All pages include interactive elements and are fully responsive.
