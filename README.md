# Moss Energy — Change Control Board (CCB) Portal

A **GitHub Pages** site hosting interactive documents and tools for the Moss Energy Change Control Board workflow (v2.1).

## 📄 Pages

| Page | Description |
|------|-------------|
| **[CCB Home](index.html)** | Central hub with document directory and CCB overview statistics |
| **[Change Request Form](01_Change_Request_Form_V3.html)** | Interactive V3 form with dynamic risk assessment, category selection, and multi-section approval workflow |
| **[Change Request Register](07_Change_Request_Register.html)** | Searchable, filterable register of all change requests with status tracking |
| **[Post-Implementation Review](12_PIR_Template.html)** | Structured PIR template for evaluating change outcomes and capturing lessons learned |
| **[Workflow Flowchart](CCB_Deliverables_15_Change_Request_Workflow_Flowchart.html)** | Detailed 7-phase change request workflow with interactive stages and quality gates |
| **[Swimlane Flowchart](17_Section_Responsibility_Swimlane_Flowchart.html)** | Section responsibility and email notification swimlane diagram |

## 🧭 Navigation

All pages include a slide-out navigation sidebar accessible via the **☰** hamburger button in the top-left corner. The current page is highlighted for orientation.

## 🚀 Deployment

This site is designed to be hosted on **GitHub Pages** with zero build steps — just static HTML files.
